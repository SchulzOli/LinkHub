# Contributing And Helping

Thanks for helping with this project.

## How You Can Help

- Fix bugs
- Request changes of any kind
- Improve and help with the project

## Before Opening A Pull Request

- Keep the change focused
- Avoid unrelated cleanup in the same pull request
- Open an issue first for larger changes
- Do not push directly to `main`
- Draft pull requests are fine

## Local Setup

```bash
npm install
npm run dev
```

Use the checks that fit your change:

```bash
npm run lint
npm run build
npm run test
npm run test:e2e
```

## Project Fit

This project aims for a minimal, board-first experience with restrained, high-contrast UI.

- Keep the canvas central
- Avoid adding unnecessary chrome
- Prefer clear, direct interactions

## Pull Requests

- Explain what changed and why
- Link the related issue when relevant
- Add screenshots for visible UI changes
- Add or update tests when behavior changes
- Mention skipped checks if they would normally be relevant
